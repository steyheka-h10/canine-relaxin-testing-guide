# Canine Relaxin Testing Guide

A comprehensive training guide for veterinary technicians and students on canine pregnancy detection using relaxin hormone blood testing.

## Purpose
This guide provides structured, evidence-based information on the physiology, protocol, interpretation, and troubleshooting of relaxin hormone testing for canine pregnancy detection. It is designed for educational purposes and onboarding of new veterinary staff.
